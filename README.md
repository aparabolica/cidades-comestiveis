## Install

Requirements:

* [Node.js](https://nodejs.org)
* [MongoDB](https://www.mongodb.org)

Clone locally:

    git clone https://github.com/codigourbano/cidades-comestiveis.git

Install dependencies:

    npm install

Start the server:

    npm start
